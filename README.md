# 🛒 ShopMate - E-Commerce Web App

ShopMate is a simple and interactive **e-commerce web application** built using **Streamlit**, **Pandas**, and **Python**. It allows users to browse products, filter them, and add items to a shopping cart.

---

## 🚀 Features

- 🏷️ Product listing from CSV dataset
- 🔍 Filter products by category
- 💰 Price range filter
- 🛒 Add to Cart functionality
- 📦 Session-based cart storage
- ⚡ Fast and lightweight UI using Streamlit

---

## 🛠️ Tech Stack

- **Frontend & Backend:** Streamlit
- **Data Handling:** Pandas, NumPy
- **Visualization (optional):** Matplotlib, Seaborn
- **Language:** Python

## 📊 Dataset

The project uses a CSV file with product details:

**Columns:**
- `id` → Unique product ID
- `name` → Product name
- `category` → Product category
- `price` → Product price
